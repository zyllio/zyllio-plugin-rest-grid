# Zyllio Plugin Angular : Rest Grid

This Zyllio plugin is a Rest Grid Component implemented using [Angular](https://angular.io/)

# Demonstration

<img src="./snapshots/demo.gif">

# Development environment

To build this plugin, Node.js 18+ is required

# Experiment

Install this pre-built file to experiment the plugin:
```
https://zyllio.github.io/zyllio-plugin-rest-grid/dist/plugin.js
```

# Build

```shell
npm install

npm run build
```

# Install

Once built, only plugin.js file is to be installed within Zyllio Studio under Designer/Plugins panel
